Проект ГПО

Основная цель:
	- Разработать систему, способную обучать студентов определённым темам с помощью индивидуального подстраивания под каждого студента, а также позволяющую преподавателю проверять знания его студентов.

Основные части системы:
	- Интерфейс для взаимодействия с пользователем;
	- Ядро системы, которое будет разрабатывать план обучения студента в зависимости от уже накопленных студентом знаний.
	- Модули, способные принимать работы от студентов, проверять их на соответствие заданию и выдавать результаты преподавателю.

Основные задачи:
	- Разработать веб-интерфейс для взаимодействия системы и пользователя;
	- Разработать серверное приложение, выполняющее функции ядра системы;
	- Наполнить базу знаний системы некоторыми выбранными темами;
	- Разработать API для проверяющих модулей;
	- Разработать некоторые проверяющие модули;
	- Наполнить базу задач для некоторых тем.

План работ:
	1) Подготовка к разработке:
		- Проанализировать различные способы обучения и их качество;
		- Разработать алгоритмы подбора статей ядром для студента;
		- Составить несколько примеров готовых статей;
	2) Начальные этапы разработки:
		- Спроектировать архитектуру ядра и модулей.
		- Разработать API для проверяющих модулей.
		- Разработать интерфейс-заглушку для упрощения дальнейшей разработки;
	3) Поздние этапы разработки:
		- Разработать ядро системы;
		- Разработать полноценный веб-интерфейс;
		- Разработать несколько проверяющих модулей по выбранным дисциплинам;
		- Провести локальное тестирование системы на небольшом количестве человек.
	4) Заполнение базы данными:
		- Наполнить базу множеством статей;
		- Заполнить базу множеством задач для проверяющих модулей.
	5) Тестирование:
		- Провести масштабное тестирование системы на относительно большом количестве человек.
